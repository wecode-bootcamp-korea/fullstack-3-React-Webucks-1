# 🍓 팀 작업 현황

- 로그인, 리스트, 디테일 페이지 레이아웃
- id, pw 입력 시 로그인 버튼 활성화 기능
- 반응형 레이아웃 적용
- 목데이터를 활용한 서버와 클라이언트 간에 통신 구현
- 아이템 별로 좋아요 버튼 클릭시 하트 색깔 변경
- 댓글 추가와 삭제 기능 구현
- 로그인 인증으로 토큰 받기 구현
